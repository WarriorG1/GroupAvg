编译命令(提前准备好input.csv数据文件)：
make avg
./avg

假设foo(id int, num int);
使用到的算法：归并排序

整体思路
1.从文件中读取并解析数据到链表中
2.对链表中的数据归并排序
3.根据排序完成的链表进行结果集运算，avg使用求差法，可以避免整形数据溢出
4.输出结果集
============================
测试
结果集同PG数据库一致
============================
性能分析
归并排序算法是一种比较稳定的排序算法，但是对于越是乱序的数据排序效率高，对于趋近于有序的数据排序效率低且内存消耗大。
优化方向
主要还是取决于实现group by的算法，是否可以考虑使用hash桶

其他方式
如果数据量较小，最大值较小且不计算内存的情况下，是否可以考虑使用键索引计数法。
首先根据num的值计算频率，再把频率转换为索引，
接下来对数据进行分类的时候需要去重得到num结果集以及计算AVG的结果集
回写的步骤可以省略
